# מטלה 5: מערכת סוכני AI (CrewAI) - mm-biu26
## תיאור המטלה
מערכת רב-סוכנים המבוססת על CrewAI לניהול תקשורת תאגידית במצבי לחץ. 
המערכת הופכת תלונות עובדים (כמו מטלות מחוץ לשעות העבודה) למיילים מנומסים ומקצועיים.

## ארכיטקטורת הסוכנים
המערכת משתמשת בשרשרת סדרתית (Sequential Process) של 4 סוכנים:
1. **Venting Analyst:** מנתח את התלונה.
2. **Spin Doctor:** בונה אסטרטגיה של תיעדוף.
3. **Buzzword Translator:** מתרגם לשפת תאגיד.
4. **HR QA:** מבצע בקרת איכות סופית.

## הוכחת ריצה
להלן צילומי המסך מהרצת הקוד בטרמינל המציגים את תהליך עבודת הסוכנים והתוצר הסופי:

![צילום מסך של תהליך הריצה](כאן_להוסיף_את_קישור_התמונה_או_לגרור_את_הקובץ)

![צילום מסך של התוצר הסופי](כאן_להוסיף_את_קישור_התמונה_או_לגרור_את_הקובץ)

## הוראות הרצה
1. התקנת ספריות: `pip install crewai`
2. הגדרת מפתח ה-API: `export GEMINI_API_KEY="your_key"` (או פקודה מתאימה ב-Windows)
3. הרצה: `python main.py`
